Imagery & Style

Photography Style

  • Authentic & Diverse: Real people, global representation
  • Dynamic Energy: Motion, action, competitive moments
  • Tech-Forward: Clean compositions, modern settings
  • Warm Lighting: Natural light, approachable yet premium

Illustration Style

  • Style: Geometric, clean lines, minimal detail
  • Color: Brand palette only (Orange, Navy, White)
  • Mood: Professional, confident, accessible

Iconography

Icon Style

  • Stroke-based icons (not filled) for UI
  • 2px stroke weight at default size
  • Rounded corners (2-4px radius)
  • 24x24px base grid

Color Usage

  • Primary: Warm White (#F5EDE6) on dark backgrounds
  • Accent: Orange (#FF6B1A) for active/selected states
  • Disabled: Gray (#8C8C8C)
